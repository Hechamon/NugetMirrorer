# This is a project that mirrors one nuget feed to another

## How to use the tool
``` text
Usage: dotnet [--source <String>] [--destination <String>] [--search <String>] [--api-key <String>] [--dry-run] [--log-level <LogLevel>] [--help] [--version]

NugetMirrorer
Options:
  --source <String>          (Required)
  --destination <String>     (Required)
  --search <String>         
  --api-key <String>         (Required)
  --dry-run                 
  --log-level <LogLevel>     (Default: Minimal) (Allowed values: Debug, Verbose, Information, Minimal, Warning, Error)
  -h, --help                Show help message
  --version                 Show version
```
